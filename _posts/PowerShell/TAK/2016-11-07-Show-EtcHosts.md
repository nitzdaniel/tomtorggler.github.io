# Show-EtcHosts

## SYNOPSIS
Display \etc\hosts file content.

## SYNTAX

```
Show-EtcHosts
```

## DESCRIPTION
This funtion gets the content of the hosts file, parses the lines and outputs 
a custom object with HostName and IPAddress properties.
