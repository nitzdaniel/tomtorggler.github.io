## SYNOPSIS 

Add an entry to local hosts file.

<!-- more -->

## SYNTAX 

```powershell
Add-EtcHostsEntry [-IPAddress] <String> [-Fqdn] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION 

Adds a lines to the \etc\hosts file of the local computer.
